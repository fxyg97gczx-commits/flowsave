# FlowSave v2.1

> 구독과 고정비, 이제 한눈에 — 가족 공유, 자동 등록, 결제 알림까지

## v2.1 새 기능

- **월 예산 한도** — 한도 설정, 진행률 바, 초과 알림
- **무료 체험 추적** — 체험 종료 D-day, D-3/1/당일 알림
- **캘린더보내기** — .ics 다운로드, Google 캘린더 연동
- **리포트 이미지 공유** — PNG 저장/공유/클립보드 복사
- **가격 변동 기록** — 인상/인하 이력 및 알림

## v2.0 기능

- **결제 알림** — D-7, D-3, 당일 앱 내 + 브라우저 푸시 알림
- **문자/이메일 자동 등록** — 카드 결제 문자 붙여넣기로 구독 자동 인식 (Premium)
- **가족·공유 관리** — Netflix, iCloud 등 공유 구독 비용 분담 계산
- **절약 인사이트** — 중복 구독, 미사용, 연간결제 전환 제안
- **월별 리포트** — 자동 생성, 전월 대비, TOP 3, 이메일보내기
- **PWA 설치** — 홈 화면 추가, 오프라인 지원
- **탭 네비게이션** — 홈 / 구독 / 가족 / 리포트 / 설정
- **JSON·CSV보내기** — Premium 데이터보내기

## 시작하기

```bash
npm install
npm run dev
```

또는 `start.bat` 더블클릭

## 광고 수익화

- **무료**: 홈·구독·리포트에 배너/네이티브 광고 (데모 또는 AdSense)
- **Premium**: 모든 광고 제거

### Google AdSense 연동 (선택)

`.env.local`에 설정:

```
NEXT_PUBLIC_ADSENSE_CLIENT_ID=ca-pub-xxxxxxxx
NEXT_PUBLIC_ADSENSE_SLOT_HOME_BANNER=xxxxxxxx
```

설정하지 않으면 금융·절약 관련 **데모 광고**가 표시됩니다.

## 프라이버시

모든 데이터는 브라우저 localStorage에만 저장됩니다. 서버 전송 없음.
